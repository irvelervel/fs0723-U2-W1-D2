POSITION: STATIC

Il valore "static" di position è anche il valore di default. Un elemento dotato di position: static non è posizionato nella pagina in alcun modo speciale, seguirà sempre il normale flow degli elementi. Può essere necessario impostarlo a mano quando un elemento eredita questa proprietà da un contenitore oppure quando è necessario differenziarne il valore in una media query.
